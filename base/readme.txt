Programming Language: Python 3.11
Framework: Djanog 5.0

How To Get Started
------------------
1. Make sure that python installed in your system

2. Now create a Virtualenv with your desired name
    command to create Virtualenv:
        virtualenv <-virtualenv_name->
    
3. Then activate the env
    command to activate virtualenv 
        <system_file_path>\<virtualenv_name>\scripts\activate

4. Then Set the file path to base folder(which is the main project folder)
    then install the requirements command
        pip install -r requirements.txt

5. Then run the project or application
    command to run the project
        python manage.py runserver

6. open the http://127.0.0.1:8000 in your browser after the succesfull installation and starting of the project
